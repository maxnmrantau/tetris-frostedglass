# Tetris Modern Minimalist (Frosted Glass Edition)

Sebuah adaptasi modern dari game klasik Tetris dengan estetika desain **Frosted Glass** yang elegan, animasi yang halus, dan efek suara sintetis yang imersif. Dikembangkan menggunakan React dan Vite.

## ✨ Fitur Utama

- **Desain Minimalis Modern**: Menggunakan tema *Frosted Glass* (Glassmorphism) dengan latar belakang *mesh gradient* yang dinamis.
- **Efek Suara Sintetis**: Audio yang dibuat secara prosedural (Web Audio API) untuk rotasi, pembersihan baris, *hard drop*, dan *game over*.
- **Ghost Piece**: Indukator visual tempat jatuhnya blok untuk membantu presisi permainan.
- **Sistem Level Progresif**: Kecepatan blok akan meningkat secara otomatis seiring bertambahnya level.
- **Responsif**: Dapat dimainkan di desktop maupun perangkat mobile dengan kontrol layar sentuh khusus.
- **Efek Glow & Animasi**: Visual yang memanjakan mata menggunakan `motion/react` (Framer Motion).

## 🚀 Teknologi yang Digunakan

- **React**: Library UI utama.
- **Vite**: Build tool super cepat untuk pengembangan frontend modern.
- **Tailwind CSS**: Framework CSS untuk styling utilitas.
- **Motion (Framer Motion)**: Untuk animasi transisi dan efek visual.
- **Web Audio API**: Untuk menghasilkan efek suara retro-futuristik langsung di browser tanpa file audio eksternal.
- **Lucide React**: Set ikon yang bersih dan konsisten.

## 🛠️ Cara Menjalankan

1. Clone repositori ini.
2. Pastikan Anda memiliki Node.js terinstal.
3. Jalankan `npm install` untuk menginstal semua ketergantungan.
4. Jalankan `npm run dev` untuk memulai server pengembangan.
5. Buka `http://localhost:3000` di browser Anda.

## 🎮 Kontrol Permainan

- **Panah Atas (↑)**: Rotasi Blok
- **Panah Kiri/Kanan (← →)**: Pindah Samping
- **Panah Bawah (↓)**: Jatuh Cepat (Soft Drop)
- **Spasi**: Jatuh Instan (Hard Drop)
- **P / Esc**: Pause / Resume
